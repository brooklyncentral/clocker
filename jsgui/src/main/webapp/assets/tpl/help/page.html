Clocker Help

This is the Clocker JavaScript web client for the Apache Brooklyn REST API.

Clocker is an Apache-licensed open-source project for application deployment and management, using Docker and Marathon in the Cloud.

You are currently using Clocker Version 1.1.0-PREVIEW.20160118

References

  • Clocker project web page
  • Clocker source on Github
  • Apache Brooklyn web page

Note you can observe the REST requests made by the Javascript GUI through the developers console in many browsers.
